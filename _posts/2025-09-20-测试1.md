### 行内公式
物理学中，质能方程 $$E=mc^2$$ 描述了质量与能量的关系。

### 块级公式
$$
\int_{-\infty}^{+\infty} e^{-x^2} dx = \sqrt{\pi}
$$

### 带编号的公式
$$
\begin{equation}
\sum_{k=1}^n k = \frac{n(n+1)}{2}
\label{eq:sum}
\end{equation}
$$
公式\eqref{eq:sum}为前n项自然数求和公式。


## 4. 代码块

### 指定文件名的代码
```python
def factorial(n):
    if n == 0:
        return 1
    return n * factorial(n-1)

print(factorial(5))  # 输出120
```
